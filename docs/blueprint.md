# **App Name**: Sai Paduka Portal

## Core Features:

- Event Details Display: Publicly display core event information including name, organizer, location, date, contact details, and an embedded Google Map of the venue.
- User Authentication: Secure user access for Darshan and Volunteer registrations via Google Authentication.
- Volunteer Registration System: A form for users to sign up as volunteers (Name, Phone No., Area of Service like Crowd Management, Prasad Distribution, Cleanliness, Medical Help) after Google Authentication. Registered volunteer data is stored and viewable in the admin panel.
- Darshan Registration: A dedicated page for users to register for Darshan, accessible only after successful Google Authentication. Registered participant data is stored and viewable in the admin panel.
- Donation & Prominent Donors Management: A frontend page displaying various donation purposes (e.g., Event Arrangements). An admin tool to edit donation purposes and manage details for a 'prominent donors' display column.
- Event Photo Gallery: A public-facing photo gallery to showcase event images. The gallery content (upload, delete images) is fully editable via the admin panel.
- Admin Control Panel: A secure administrative interface to manage event details, donation purposes, prominent donors, photo gallery, venue address (Google Map API), and to review all registered volunteers and Darshan participants.

## Style Guidelines:

- A light color scheme centered on a spiritual saffron-gold hue. The primary color is a warm saffron orange-gold: #E5891A. The background is a very pale, warm off-white to maintain a clean devotional feel: #F9F6F0. An accent color, a deeper red-orange for important calls to action and highlights, is: #CC1E0A.
- For a clean and warm devotional aesthetic, 'PT Sans' (humanist sans-serif) will be used consistently for all headlines and body text, ensuring excellent readability and a modern, personable feel.
- Utilize simple, clean, and symbolic iconography that subtly references spiritual themes and event activities, without being overly ornate.
- Emphasize a clean, spacious, and responsive layout, ensuring optimal readability and user experience across all devices. A clear visual hierarchy will guide users through event information and registration processes.
- Focus on subtle, performant transitions and micro-interactions that enhance user feedback without hindering fast loading times or SEO. Avoid heavy animations.
